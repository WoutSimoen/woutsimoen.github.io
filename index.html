Selected Projects

YeastNet Protein Network Analysis

A computational tool for parsing and analyzing protein-protein interaction networks. This project utilizes spectral graph theory to evaluate network topology and identify functional protein clusters.

  • Implemented Graph Laplacian matrices to identify connected components.
  • Developed algorithms for clustering coefficient calculation and clique detection.
  • Optimized for high-confidence biological datasets (YeastNet v3).
